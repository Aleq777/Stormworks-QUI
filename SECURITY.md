# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.2.0-alpha | :yellow_square: |
| 0.1.0-alpha | :x: |

## Reporting a Vulnerability

Report issues via:
- **Private Security Advisory** - [Click here](https://github.com/aleq777/Stormworks-QUI/security/advisories)
- **Discord QUI Server** - [Invitation link]()

Your report will be answered as soon as I will be able to read it.
- If your report will be confirmed - I'll fix it and release a new version.
- If your report will be denied - I'll send you reasons for that and alternatives for fixing your problem.
